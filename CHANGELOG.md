# Changelog

All notable changes to this project will be documented in this file.

## 0.0.1

- Initial preview release of Accessor Discovery
- Static discovery for supported xarray and pandas accessors
- Completion, hover, signature help, and definition overlays for discovered accessors
- Recursive import analysis from the active Python file
- Output-channel report for debugging discovered accessors and unresolved imports
- Definition support for accessor namespaces, members, and resolved import namespaces
